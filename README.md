# Python-Mini-Projects
A collection of beginner-friendly Python mini projects showcasing core programming concepts and problem-solving skills. 

## 1-Password Checker

A simple Python program that simulates a password verification system. Users have three attempts to guess the correct 
password, with a hint provided before the first attempt. The program locks access after three incorrect guesses.
## 2. Rock Paper Scissors

A two-player command-line Rock Paper Scissors game featuring input validation, case-insensitive input, winner determination, and draw handling. Built to practice conditional logic, user input, and basic Python programming concepts.
